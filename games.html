Interactive Therapeutic Tools

Therapeutic Games

Interactive tools for emotional learning, coping skills practice, and therapeutic engagement

Using Games in Therapy

Games offer a low-pressure way to explore emotions, practice coping strategies, and build therapeutic rapport. These browser-based tools require no downloads or internet connection and can be used during sessions, assigned as between-session activities, or integrated into group work. Each game targets specific therapeutic skills and includes age recommendations to guide clinical use.

🃏

Feelings Memory Match

Classic memory card game with 12 emotion pairs. Match faces and reflect on personal experiences with each emotion as you play.

Ages 5-12 Emotion Identification
🎲

Coping Skills Spinner

Spin the colorful wheel to discover coping strategies. Each spin lands on a technique with a brief explanation of how to practice it.

All Ages Coping Skill Exploration
🧠

Thought Challenger

Interactive CBT exercise. Identify cognitive distortions, rate belief strength, and practice writing balanced alternative thoughts.

Ages 13+ Cognitive Restructuring
🌬️

Breathing Guide

Guided breathing with an animated circle. Choose from Box Breathing, 4-7-8, or Simple Deep Breathing with calming visual transitions.

All Ages Relaxation & Regulation
💗

Feelings Check-In

Quick mood check-in tool. Select your current feeling, rate its intensity, and receive a tailored coping suggestion. Great for session openers.

All Ages Emotional Awareness

For Clinicians

📚 Integrating Games Into Sessions

  • Use Feelings Memory Match as a warm-up to build emotional vocabulary before deeper work
  • The Coping Skills Spinner works well as a session closer to identify a take-home strategy
  • Thought Challenger pairs naturally with CBT psychoeducation on cognitive distortions
  • Breathing Guide can anchor regulation practice at any point in a session
  • Open sessions with a Feelings Check-In to set the therapeutic agenda

📝 Documentation Tips

  • Note which emotions the client could and could not identify in Memory Match
  • Record the client's preferred coping strategies from the Spinner for treatment planning
  • Track belief rating changes in Thought Challenger as a measure of cognitive flexibility
  • Document breathing pattern tolerance and duration for progress monitoring
  • Use Check-In responses to track mood trends across sessions

🎯 Suggested Populations

  • Memory Match: Children with alexithymia, ASD, or limited emotional vocabulary
  • Spinner: Clients new to therapy or building a coping toolkit
  • Thought Challenger: Adolescents and adults with anxiety, depression, or negative self-talk
  • Breathing Guide: Anxiety, PTSD, panic, or any client learning regulation skills
  • Check-In: All clients; especially helpful for those who struggle to verbalize feelings
